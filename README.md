# 绘图自动机

## 使用

安装 NodeJS

``` bash
git clone https://github.com/Enter-tainer/luoguPaintAutomatonTS
```

``` bash
npm install
```

``` bash
tsc
```

``` bash
cd dist
```

``` bash
node index.js
```
## 图片

图片用luogu绘版所使用的32进制数表示，保存在 `data/test.32` 中，需确保图片为矩形。

请将 cookies 放置在 `users.json` 中

在 `index.ts` 中可以调整图片左上角坐标
